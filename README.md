#JEC Sample Projects

The JEC Sample Projects is a part of the [JavaScript Entreprise Container][jec-url] projects.

[![][jec-logo]][jec-url]



## Requirements

The BLM project needs the following system parameters in order to work correctly:

- JEC 1+ _(GlassCat)_
- Node 6+
- npm 3+
- TypeScript 2+
